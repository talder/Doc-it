# Distraction-Free Mode

Distraction-free mode hides the sidebar, topbar, and document bar, leaving only the editor and a slim control bar at the top. It is ideal for focused writing sessions.


---

## Entering Distraction-Free Mode

Click the **PenLine** (distraction-free) button in the document actions area. You do not need to be in edit mode first — clicking the button activates edit mode and distraction-free simultaneously.

If another user is currently editing the document, a presence warning is shown instead.

---

## The Slim Bar

While in distraction-free mode, a thin bar at the top of the screen shows:

| Element | Description |
|---|---|
| Document title | Name of the open document |
| Autosave status | "Autosave on" or "Saving…" |
| Discard button | Restore the document to its pre-edit state |
| Done button | Save and exit editing (creates a revision) |
| Minimize icon | Exit distraction-free mode but stay in edit mode |

---

## Exiting

Three ways to exit:
1. Press **Esc** — exits distraction-free mode and returns to normal edit mode.
2. Click the **Minimize** icon — same as Esc.
3. Click **Done** — finishes editing and returns to the full workspace view.
